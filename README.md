# 🎭 הקרנה לתפאורה

אפליקציית Projection Mapping לתפאורה ולאמנות — Next.js 14, WebGL, Supabase.

## Stack
- **Frontend:** Next.js 14 + React + TypeScript + Tailwind CSS
- **Canvas Engine:** WebGL + Canvas API
- **State Management:** Zustand
- **Backend/DB:** Supabase
- **Deployment:** Vercel

## פיצ'רים
- ✅ Canvas עם Projection Mapping
- ✅ Corner Pin / Warp
- ✅ Mask & Cut tools
- ✅ שכבות (Dual Channels)
- ✅ יבוא וידאו ותמונות
- ✅ אפקטים בזמן אמת
- ✅ פלט חיצוני לפרויקטור
- 🔜 Audio Reactive
- 🔜 MIDI Control
- 🔜 שמירת פרויקטים בסופאבייס

## התחלה
```bash
npm install
npm run dev
```
