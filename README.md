# 🎭 הקרנה לתפאורה

אפליקציית Projection Mapping לתפאורה — Next.js 14, WebGL, Supabase.

## Stack
- **Frontend:** Next.js 14 + React + TypeScript + Tailwind
- **Canvas:** WebGL + Canvas API
- **State:** Zustand
- **Backend:** Supabase
- **Deploy:** Vercel

## התחלה
```bash
npm install
npm run dev
```
